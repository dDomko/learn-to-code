HTML

Čo je HTML?

  • HTML je skratka pre Hyper Text Markup Language

  • HTML je štandardný značkovací jazyk na vytváranie webových stránok

  • HTML popisuje štruktúru webovej stránky

  • HTML pozostáva zo série prvkov

  • Prvky HTML hovoria prehliadaču, ako má zobraziť obsah

  • Prvky HTML označujú časti obsahu, ako napríklad "this is a heading", "this is a paragraph", "this is a link", atď.

    • CSS

    Čo je CSS?

  • CSS je jazyk, ktorý používame na úpravu štýlu webovej stránky.

  • CSS je skratka pre Cascading Style Sheets

  • CSS popisuje, ako sa majú prvky HTML zobraziť na obrazovke, papieri alebo v iných médiách

  • CSS ušetrí veľa práce. Dokáže ovládať rozloženie viacerých webových stránok naraz

  • Externé šablóny so štýlmi sú uložené v súboroch CSS
